About ToyCraft Tales

ToyCraft Tales is a data visualization project built using Tableau to explore toy manufacturer trends across US states from 2005 to 2016.

The dataset was sourced from Kaggle and analyzed to uncover growth patterns, geographic distribution, and industry insights.

Technologies Used:

  • Tableau
  • Flask
  • HTML & CSS
